# Quick Start Guide

## 🚀 5-Minute Setup

### Option 1: Using Local MongoDB

#### Step 1: Install MongoDB
- **Windows**: Download from https://www.mongodb.com/try/download/community
- **Mac**: `brew install mongodb-community`
- **Linux**: Follow official MongoDB installation guide

#### Step 2: Start MongoDB
- **Windows**: MongoDB runs as a service automatically
- **Mac/Linux**: Run `brew services start mongodb-community`
- **Verify**: Open MongoDB Compass

#### Step 3: Backend Setup
```bash
cd "C:\Users\user\Desktop\Car Project\backend"
npm install
npm start
```
✅ Backend ready at `http://localhost:5000/api`

#### Step 4: Frontend Setup (New Terminal)
```bash
cd "C:\Users\user\Desktop\Car Project\frontend"
npm install
npm start
```
✅ Frontend ready at `http://localhost:3000`

---

### Option 2: Using MongoDB Atlas (Cloud)

#### Step 1: Create MongoDB Atlas Account
1. Go to https://www.mongodb.com/cloud/atlas
2. Sign up (free tier available)
3. Create a cluster

#### Step 2: Get Connection String
1. Click "Connect"
2. Choose "Connect your application"
3. Copy connection string
4. Replace `<username>`, `<password>`, `<database>`

#### Step 3: Update Backend `.env`
```
PORT=5000
MONGODB_URI=mongodb+srv://user:password@cluster.mongodb.net/car-selling-db
NODE_ENV=development
```

#### Step 4: Run Backend & Frontend (as above)

## 🔄 Data Flow Example

### Creating a Car Listing

```
1. User fills form on AddCarPage (React)
        ↓
2. Clicks "Create Listing" button
        ↓
3. Form data validated in AddCarPage.jsx
        ↓
4. API call: carAPI.createCar(formData)
        ↓
5. Axios sends POST request to http://localhost:5000/api/cars
        ↓
6. Express receives request in server.js
        ↓
7. Routes to carController.createCar()
        ↓
8. Controller validates data
        ↓
9. Controller calls Car.create() (MongoDB)
        ↓
10. MongoDB saves document and returns _id
        ↓
11. Controller returns response with created car
        ↓
12. Frontend receives response, shows success message
        ↓
13. User redirected to HomePage
        ↓
14. HomePage fetches all cars and displays new one
```

## 🧠 Understanding the Code

### Backend - How a Request Works

**File**: `server.js`
```javascript
// 1. Express app starts and connects to MongoDB
app.use(cors())  // Allow frontend to make requests
app.use(express.json())  // Parse JSON bodies
app.use('/api/cars', require('./routes/carRoutes'))  // Route requests
```

**File**: `routes/carRoutes.js`
```javascript
// 2. Route matches GET /api/cars to controller function
router.get('/', carController.getAllCars)
```

**File**: `controllers/carController.js`
```javascript
// 3. Controller function executes
exports.getAllCars = async (req, res) => {
  // Get all cars from database where isSold = false
  const cars = await Car.find({ isSold: false })
  // Send back as JSON
  res.json({ success: true, data: cars })
}
```

**File**: `models/Car.js`
```javascript
// 4. Model defines the structure
const carSchema = new Schema({
  title: { type: String, required: true },
  // ... other fields
})
module.exports = mongoose.model('Car', carSchema)
```

---

### Frontend - How a Component Works

**File**: `pages/HomePage.jsx`
```javascript
useEffect(() => {
  // 1. When component mounts, fetch cars
  fetchCars()
}, [])

const fetchCars = async () => {
  // 2. Make API call
  const response = await carAPI.getAllCars()
  // 3. Store result in state
  setCars(response.data.data)
}

// 4. Render cars using map
return (
  <div className="grid grid-cols-3">
    {cars.map(car => <CarCard key={car._id} car={car} />)}
  </div>
)
```

## 🚨 Common Error Solutions

### Error: "Cannot find module 'express'"
```bash
# Solution: Install dependencies
cd backend
npm install
```

### Error: "MongooseError: Cannot connect to MongoDB"
```bash
# Solution 1: Start MongoDB
mongod

# Solution 2: Update MONGODB_URI in .env
# Make sure it's correct
```

### Error: "CORS error: Access denied"
```bash
# This should be fixed in server.js, but if not:
# Make sure backend server.js has:
const cors = require('cors')
app.use(cors())
```

### Error: "Cannot GET /api/cars"
```bash
# Solution: Ensure backend server is running
# Check terminal where you ran "npm start"
# Should show "Server running on port 5000"
```
